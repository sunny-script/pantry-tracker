# Pantry Tracker

A sleek and intuitive pantry management app that helps users **track, organize, and manage** their pantry items efficiently. Designed with a modern UI and simple local storage for easy use.

## Deployment
Deployed with Vercel, check it out! 

[PantryPro](https://pantry-tracker-six.vercel.app/)

## Features

**Add Pantry Items** ‚Äì Input item name, select status (Low, Moderate, Excess), and choose a category.  
**Local Storage** ‚Äì Saves pantry items so they persist after refreshing.  
**Search & Filters** ‚Äì Search by name, filter by status or category.  
**Edit & Delete** ‚Äì Modify or remove pantry items directly in the table.  
**Modern UI** ‚Äì Styled using Tailwind CSS for a clean and professional look.
